# Networking Fundamentals

A computer network is a group of devices connected to share data and resources.

## Common Network Types
- LAN (Local Area Network): Home, office, school
- WAN (Wide Area Network): Internet

## Client and Server
- Client: Requests data (browser, app)
- Server: Provides data (web server, DNS)

## Cybersecurity View
Most cyber attacks happen through networks, so understanding networking is critical.
